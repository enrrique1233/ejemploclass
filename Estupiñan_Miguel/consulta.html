Qué es Git?

Git es un software de control de versiones, pensando en la eficiencia y la confiabilidad
mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos
de código fuente.

Beneficios

1. Flujo de trabajo en la rama de función
2. Desarrollo distribuido
3. Solicitudes de extracción
4. Comunidad
5. Ciclo de liberación más rápido

Comandos

Git clone
realiza una copia idéntica de la última versión de un proyecto en un repositorio y la guarda en tu ordenador.
git clone -> https://link-con-nombre-del-repositorio.

Git branch
Podemos usar el comando git branch para crearlas, listarlas y eliminarlas.
git clone ->https://link-con-nombre-del-repositorio.

Git checkout
Para cambiarte de una rama a otra. También lo podemos usar para chequear archivos y commits..
git clone ->git checkout :nombre-de-la-rama.

Git Porqué usar

Reduce considerablemente los tiempos de deploy (despliegue) de un proyecto, al subir solamente los cambios
son dos herramientas complementarias que ofrecen grandes funcionalidades y ventajas a la hora de llevar a cabo
proyectos de software. Permiten llevar controlar totalmente el estado de los diferentes archivos de código fuente
a lo largo del tiempo, facilitando la revisión de código, la depuración de errores y la realización de todo tipo
de pruebas.
